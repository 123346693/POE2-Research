# 角色优化器设计

角色优化不能只问“哪个装备 DPS 高”。它要先判断这个角色为什么死、为什么没伤害、以及预算买什么最划算。

## 输入

- 角色快照：职业、升华、等级、主技能、装备、抗性、生命/ES、防御层。
- 预算：例如 5D、20D、100D。
- 玩法目标：刷图、Boss、HC、SSF、速刷、开荒。
- 价格快照：每个候选升级的当前价格、样本量、来源、时间。

## 输出

- 优先级排序。
- 目标槽位和词缀。
- 预计花费。
- 预计伤害/生存收益。
- 置信度。
- 交易搜索方案。

## 默认原则

- 抗性没满时，先不推荐昂贵伤害装。
- 低有效血量时，先补主防御槽。
- 武器明显落后时，武器通常是攻击 Build 的第一伤害升级。
- 每条建议都必须能在价格层找到依据；找不到就降低置信度。
